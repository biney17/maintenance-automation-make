# Maintenance Automation - Python + GPT

Système automatisé de maintenance industrielle utilisant Make, Python et GPT.

## Features

- Détection automatique des anomalies
- Analyse temperature, pressure, vibration
- Génération de rapports avec GPT
- Intégration Airtable
- Exécution automatique

## Installation

1. Make.com account
2. Airtable base
3. OpenAI API key
